Power BI - Externe Tools

Beschreibung: Power BI Desktop bietet eine leistungsstarke Möglichkeit, externe Tools in ein eigenes Menüband zu integrieren, um Workflows zu optimieren, Metadaten zu verwalten oder spezifische Analysen durchzuführen.

Für das Management von externen Tools können Sie folgende Anwendungen nutzen:

  • Power BI Sidetools (kostenlos) — Download
  • Conductor Pro (kostenpflichtig) — Mehr Infos

Arten von Externen Tools

Tool-Typ Beschreibung Beispiel
PowerShell-Skripte Automatisierung von Prozessen oder API-Abfragen für Power BI powershell.exe -File "C:\Scripts\mein-script.ps1"
Webseiten (URLs) Direkte Links zu Dokumentationen, Dashboards oder Web-APIs https://powerquery.how/
EXE-Dateien Lokale Anwendungen wie DAX Studio oder eigene Programme "C:\Program Files\MyTool\MyApp.exe"

Voraussetzungen

  • Power BI Desktop (aktuelle Version)
  • Power BI Sidetools installiert — Download
  • Administratorrechte, um Tools im Power BI-Verzeichnis zu registrieren
  • Ein Skript, eine Webseite oder eine Anwendung, die als externes Tool hinzugefügt werden soll

Vorgehensweise

Installation von Power BI Sidetools

  • Power BI Sidetools herunterladen: https://thebipower.fr/index.php/power-bi-sidetools/
  • Installation ausführen und sicherstellen, dass Power BI Desktop installiert ist.
  • Power BI Sidetools starten und die Benutzeroberfläche öffnen.

Neues Tool hinzufügen

  1. Klicke auf "New" oder nutze den Wizard für eine schrittweise Erstellung.
  2. Trage folgende Informationen ein:
    • Tool Name: Der Name des Tools, das in Power BI angezeigt wird.
    • Beschreibung: Optionaler Beschreibungstext.
    • Version: Falls gewünscht, kann eine Versionsnummer angegeben werden.
    • Pfad: Der Pfad zur ausführbaren Datei, PowerShell-Skript oder URL.
    • Arguments: Zusätzliche Parameter, falls notwendig.
    • Icon: Ein Bild (PNG) für das Tool hochladen.
  3. Speichere die Konfiguration mit "Save".

Test in Power BI Desktop

  1. Starte Power BI Desktop.
  2. Öffne einen bestehenden Bericht oder erstelle ein neues Modell.
  3. Navigiere zur Registerkarte "Externe Tools".
  4. Dein neu erstelltes Tool sollte hier erscheinen.
  5. Klicke auf das Tool, um es zu testen.

Bearbeiten oder Entfernen eines Tools

Falls Änderungen notwendig sind:

  1. Wähle das Tool in Power BI Sidetools aus.
  2. Klicke auf "Edit", um Anpassungen vorzunehmen.
  3. Falls das Tool nicht mehr benötigt wird, kann es über "Delete" entfernt werden.

Beispiel: Externes Tool für PowerQuery.how

Ein Beispiel für ein Web-Tool, das direkt in Power BI integriert wird:

Einstellung Wert
Name "PowerQuery How"
Beschreibung "Die Webseite powerquery.how bietet umfassende Informationen zur M-Sprache."
Version "1.0.0"
Pfad https://powerquery.how/
Icon Hochgeladen

➡️ Das Tool erscheint nun in der "Externe Tools"-Leiste in Power BI.

Zusatzfunktionen in Power BI

Ergebnis

Nach erfolgreicher Einrichtung kann das neue externe Tool direkt in Power BI Desktop verwendet werden. Vorteile:

  • Einfache Integration eigener Tools ohne manuelle JSON-Bearbeitung
  • Unterstützung von PowerShell-Skripten, Webseiten & Anwendungen
  • Personalisierung mit eigenem Logo & Beschreibung

Durch Power BI Sidetools wird die Erstellung externer Tools enorm erleichtert.
